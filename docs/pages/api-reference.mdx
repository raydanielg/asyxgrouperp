# API Reference

This section documents the internal route structure and key controller methods of the ASYX Group ERP.

## Route Structure

All admin routes are prefixed with `/admin` and require authentication.

### Public Routes

| Method | URI | Description |
|--------|-----|-------------|
| GET | `/` | Landing page |
| GET | `/login` | Login form |
| POST | `/login` | Authenticate |
| GET | `/careers/jobs` | Public job listings |
| GET | `/careers/jobs/{id}/apply` | Application form |
| POST | `/careers/jobs/{id}/apply` | Submit application |

### Admin Dashboard

| Method | URI | Description |
|--------|-----|-------------|
| GET | `/admin/dashboard` | Main admin dashboard |
| GET | `/admin/role-dashboard` | Role-specific dashboard |
| GET | `/admin/reports` | Reports page |

# Project Management

Manage all company projects with task assignment, time tracking, and bug management.

## Projects

**Features:**
- Project creation with milestones
- Team assignment
- Status tracking: Planning → In Progress → On Hold → Completed
- Budget tracking
- File attachments
- Progress percentage
- Client association
- Deadline management

**Navigation:** Admin → Projects → Projects

## Project Tasks

Break projects into actionable tasks.

**Features:**
- Task creation and assignment
- Priority levels (low, medium, high, urgent)
- Due date tracking
- Status: To Do → In Progress → Review → Done
- Task dependencies
- Time estimation vs actual

## Timesheets

Track time spent on projects and tasks.

**Features:**
- Daily time entries
- Project and task association
- Billable vs non-billable hours
- Weekly/monthly summaries
- Timesheet approval workflow
- Export for invoicing

**Navigation:** Admin → Projects → Timesheets

## Bug Tracking

Log and resolve project bugs.

**Features:**
- Bug reporting with severity levels
- Assignment to developers
- Status workflow: Open → In Progress → Testing → Closed
- Priority classification
- Screenshot/file attachments
- Resolution tracking

**Navigation:** Admin → Projects → Bugs

## Converting Proposals to Projects

The system supports converting accepted sales proposals directly into projects:

1. Navigate to Sales Proposals
2. Find an accepted proposal
3. Click "Convert to Project"
4. Project is created with:
   - Client from proposal
   - Budget from proposal value
   - Linked reference to original proposal
